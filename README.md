# カチウマ 🐴

**価値馬（＝期待値の高い馬）を見つける、個人用の競馬分析ツール。**

netkeiba から集めたデータをもとに、各馬の推定勝率と実オッズを突き合わせ、
「市場が過小評価している馬（妙味）」を期待値（EV）で可視化します。
**馬券の購入は手動**で行う前提の、分析・意思決定支援ツールです。

---

## これは何？

- レースごとに、各馬の **推定勝率・市場想定勝率・期待値・印** を表示
- 「**なぜその印なのか**」の根拠（効いた要素）まで提示
- 過去データでの **回収率・的中率（バックテスト）** を確認できる
- スマホで見られる Web アプリ（閲覧専用・個人利用）

## これは何ではない？

- ❌ 馬券の自動購入ツールではありません（購入は自分で行います）
- ❌ 必勝ツールではありません（競馬は控除率20〜30%。期待値はあくまで判断材料）
- ❌ 商用・収益化目的ではありません（個人利用）

## 仕組み（全部クラウド・無料）

```
[GitHub Actions（定期実行）]
   収集(scraper) → 分析(analysis) → predictions.json をコミット
                                          ↓ push
[Vercel] web/ を自動デプロイ → スマホで閲覧
```

開発は **このリポジトリ＋AI（ループエンジニアリング）** で進めます。
詳しくは `INDEX.md` を参照。
